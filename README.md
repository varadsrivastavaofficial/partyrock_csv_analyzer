# PartyRock CSV Analyzer

An AI-powered CSV analysis tool built using AWS PartyRock that helps users quickly understand the structure, quality, and purpose of their datasets.

**Live Application:**
https://partyrock.aws/u/VaradSrivastava/ALqbkTLF6/Excel-Data-Structure-Analyzer

---

## About This App

Upload a CSV file (exported from Excel) to receive an AI-powered analysis of its structure and contents. The AI automatically identifies column headers, explains data types, highlights patterns, and provides insights into the overall purpose of your data.

Whether you're a student, researcher, analyst, or business professional, this tool helps you understand unfamiliar datasets within seconds.

### Exporting Excel Files to CSV

1. Open your Excel file.
2. Click **File → Save As**.
3. Select **CSV (Comma delimited) (*.csv)**.
4. Save the file.
5. Upload it to the application.

## Features

### Automated Dataset Analysis

* Identifies column names and their likely meanings
* Detects numerical, categorical, textual, and date fields
* Recognises data patterns and relationships
* Summarises dataset structure

### Data Quality Insights

* Highlights potential inconsistencies
* Detects missing or incomplete information
* Identifies unusual patterns in the data

### Business Context Detection

* Infers the likely purpose of the dataset
* Provides contextual explanations
* Suggests possible analytical use cases

### No-Code Experience

Simply upload a CSV file and receive AI-generated insights without writing any code.

| Feature                       | PartyRock CSV Analyzer | Generic Chatbot             |
| ----------------------------- | ---------------------- | --------------------------- |
| Upload-and-analyse workflow   | ✅                      | ❌                           |
| Automatic schema detection    | ✅                      | Limited                     |
| Dataset-specific insights     | ✅                      | Depends on prompts          |
| Consistent analysis format    | ✅                      | Variable                    |
| Data structure interpretation | ✅                      | Manual explanation required |
| Purpose-built for CSV files   | ✅                      | General-purpose             |

Unlike a traditional chatbot, this application is specifically designed for structured data analysis. Users do not need to describe their datasets manually—the AI performs the analysis automatically.
